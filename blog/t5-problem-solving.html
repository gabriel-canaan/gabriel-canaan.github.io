Map

Map is similar to a copy of of a list, but one that has been subjected to specified changes. The original remain unchanged but the mapped copy will feature the modifications i.e. 1,2,3 times two will be mapped to 2,4,6 .

Filter

Filter is not dissimilar to a map in that it returns a copy of an original list. It’s difference lies in the fact that it returns only the item that pass a certain criteria. A list of numbers from one to one hundred could be ‘filtered’ to only return even numbers for example.

Reduce

Reduce is a bit like stock. You chuck loads of stuff in a big pot of water and boil the crap out of it. At the end you have stock, all the initial ingredients are still there (the goodness anyway) but you have a unique product. It looks similar to just adding up but there’s a lot more going on.
