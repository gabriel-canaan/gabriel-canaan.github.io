Context.

Scope in Javascript refers to the current context of the code. That is, the visibility and accessibility of the variables that are defined. Variables are defined either globally or locally.

Global.

A global declaration is one that is pertains to the entire program and is identified separately from any specific function, often at the beginning of the code.

Are you local ?

A local variable conversely, is declared inside a particular function and is valid only within the parameters of that function.

Lifespan.

The lifetime of a JavaScript variable starts when it is declared and local variables are deleted when the function is completed. When globally declared, a variable will remain accessible throughout the program’s activation.
