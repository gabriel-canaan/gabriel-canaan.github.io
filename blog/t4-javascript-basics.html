Accessing info from Arrays and Ojects
Listing items as arrays or objects seems ,superficially, to be quite similar. I’m sure we will encounter the real differences soon (hope so). What is evidently different is the way we can access the data in these ‘lists’. To obtain information from an array we identify is position in the ‘list’, that is from first [0] to last [nth] and that item can be displayed. An object, as it has properties associated with it, when ‘called’ by it’s name will display it’s attributions.

How useful are functions right ?
A Function is a bit like a mini program. A certain number of commands are initiated when various criteria are meet. Once defined it can be run again and again without the need to re write, saving the poor coder from RSI. and can be call(ed) upon at any time by entering fresh data.
